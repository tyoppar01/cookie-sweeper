
# Cookie Sweeper

Cookie Sweeper is a Minesweeper-inspired game with a cookie twist!

## Features
- Flag tiles you suspect contain bombs
- Reveal tiles to see how many cookies are nearby
- Win by flagging all cookie tiles
- Lose if you reveal a bomb

## How to Play
1. Left-click to reveal a tile.
2. Right-click to flag a tile as a cookie.
3. The number on a revealed tile shows how many cookies are adjacent.
4. Flag all cookies to win!
5. Revealing a cookie ends the game.
